# HOPPER — future projects

Where else the **emergent-skills-AI** pattern (TIGRIS / CERES / LUCIA shape) could
be pointed. Each entry: shape, why it fits, what the gap-finding would look like.

---

## Strongest fits — TIGRIS-shape (corpus → dimensions → gaps → design)

### 1. Cocktails / spirits
**Shape**: TIGER BEAT for drinks. ~10,000 known cocktails as corpus, ~12–15
axes (intensity, complexity, spirit family, occasion, glass, prep difficulty,
acid/sugar balance, ABV).
**Parliament**: Trader Vic, Audrey Saunders, Ryan Chetiyawardana, Dale DeGroff,
Ada Coleman.
**Gap example**: bitter-zero serves, low-ABV food-pairing, alcohol-free that's
genuinely great (under-explored).
**Why now**: cleanest TIGRIS-clone. Fast build. Shareable output.

### 2. Architecture / built environment
**Shape**: corpus of houses, civic buildings, libraries, third-places. ~12
axes (site, materiality, scale, public/private, climate response, social grain).
**Gap example**: climate-conscious civic library at 5,000-population scale.
**Why interesting**: design space is vast and visibly under-mapped.

### 3. Programming languages / DSLs
**Shape**: corpus of languages scored on ergonomics, type system, FFI,
runtime model, error handling, syntax cost.
**Personas**: Wadler, Hejlsberg, Stroustrup, Pierce, Hickey, Aho.
**Gap example**: ergonomic systems language with first-class effects.

### 4. Recipes / cuisines (dish-level)
**Shape**: score dishes across umami / acid / fat / texture / heat / region /
fermentation / time. Find under-explored crossovers.
**Gap example**: Andean × Levantine, Andean × Japanese fermentation.

## Strong fits — CERES-shape (equipment + economic simulation)

### 5. Hand tools
Natural extension of CERES. Plane, saw, chisel, lathe — historical functional
spec → modern redesign → financial pencil-out across solo woodworker / 5-person
shop / co-op contexts.

### 6. Coffee shops / third-place design
Catalog of variants × settlement scale × economic model. Pass condition:
financial viability + social-fabric impact.

### 7. Educational curricula by subject × age × pedagogy
Score depth/breadth, prerequisite chains, assessment fit. **Gap example**:
discrete math for 8-year-olds, project-based, no calculation drill.

---

## Strong fits — LUCIA-shape (narrative production at scale, voice-first)

### 8. Rituals / ceremonies across cultures
Extension of LUCIA at the ritual unit. From-inside framing. Voice-spectrum
machinery applies directly.

### 9. Story structures at sub-novel scale
Short story arcs, episode arcs, podcast-episode structure. Score corpus of
~500 story arcs across narrative dimensions.

## Underexplored / high-novelty

### 10. Festival / event design
Corpus of events. Dimensions: duration / density / communality / improvability.
Goal: formula for memorable.

### 11. Sound design / synth patches
Corpus of patches. Dimensions: envelope shape, modulation depth, spectral
focus, polyphony, headroom. **Gap example**: warm pad with motion under
12 voices.

### 12. Newsletter / Substack format design
Cadence × depth × voice × topic. Gap-finding for "long-form weekly
research-reportage in domain X."

### 13. Open source project structures (meta)
Repo layouts, governance, doc patterns, contributor onboarding. What makes
projects with N stars sustain past year 3.

### 14. Therapy / wellness modalities
Sensitive but ripe. Score by claim space, evidence base, time commitment,
target conditions.

## Picking next

If picking one repo to start: **cocktails**. Cleanest TIGRIS-clone, real
personas to simulate, parliaments are entertaining, gaps are real (alcohol-free
that's actually great), build is fast, output is shareable. Doesn't compete

## How an entry graduates from this list

1. The pattern fit is clear — what's the corpus, what's the parliament, what
   does gap-finding look like.
2. The user has time and motivation to start a workshop.
3. Spin up `C:\src\<slug>\`, install the standard skills (panel, ideate,
   amendment, handoff), seed an initial corpus, run the first parliament.
4. Move the entry from this hopper to a separate `## Active` section, link to
   the new repo.
