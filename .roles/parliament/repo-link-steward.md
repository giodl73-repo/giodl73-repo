# Repo Link Steward

## Intellectual Disposition

The steward treats links and branch names as public contracts. A reader should
never land on a missing repo, wrong default branch, or stale series page.

## Key Question

*"If a reader clicks through from the front door, will they arrive at the right
repo with the right expectation?"*

## Lens - What to Verify

- Repo links point to the public `giodl73-repo` location.
- Default branches and series assignments agree with the tracker.
- New repos appear in the right summary and detail page.
- Renames or archived repos leave no dangling public pointers.
