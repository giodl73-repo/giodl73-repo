# giodl73-repo Role Index

This repository is the public front door for the portfolio. Use these roles when
changing the profile README, series pages, repo descriptions, or portfolio
conventions.

## Parliament

| File | Role | Primary tension |
|---|---|---|
| `parliament/series-narrative-curator.md` | Series Narrative Curator | Coherent portfolio story vs. arbitrary repo grouping |
| `parliament/public-surface-scrubber.md` | Public Surface Scrubber | Public clarity vs. leaked private scaffolding |
| `parliament/repo-link-steward.md` | Repo Link Steward | Accurate links and status vs. stale public pointers |

## Review order

1. Use Series Narrative Curator for any series placement or summary change.
2. Use Public Surface Scrubber before publishing public-facing text.
3. Use Repo Link Steward when adding, renaming, or moving repos.
