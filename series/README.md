# Series

The public repos are grouped by series. A series is a product line, research
line, or reusable infrastructure family with a shared story. Across the series,
Rust is the implementation default for reusable CLIs, libraries, verifiers, and
engines.

| Series | Public promise | Detail page |
|--------|----------------|-------------|
| AI Methodology | Evidence trails and simulated expert review. | [ai-methodology.md](ai-methodology.md) |
| Games Design | Playable design workshops, simulation, and shared experience infrastructure. | [games-design.md](games-design.md) |
| Knowledge Systems | Structured reference, history, place, and production corpora. | [knowledge-systems.md](knowledge-systems.md) |
| Design Labs | Visualization, writing, music, and media design practice. | [design-labs.md](design-labs.md) |
| Election Systems | District construction, reusable plan packages, count evidence, and audit replay. | [election-systems.md](election-systems.md) |
| Standards & Protocols | Portable identifiers, transfer formats, and repository conventions. | [standards-protocols.md](standards-protocols.md) |
| Tools & Infrastructure | Compilation, context selection and algebra, harness replay, data acquisition, and graph kernels. | [tools-infrastructure.md](tools-infrastructure.md) |
| Applied Systems | Evidence-heavy workbenches for sports and public infrastructure. | [applied-systems.md](applied-systems.md) |
